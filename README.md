# Kotlin REPL

A simple implementation of a Kotlin REPL.

Wanted to figure out how to get an embedded Kotlin REPL running,
so did some Googling and created this project.

To run:

* Clone repo
* cd into folder
* run ./gradlew --console=plain run

You're welcome..
